# Logging

This section contains everything you need for logging in and accessing your Copilot Studio POC environment.

 
<br>

<h1>üë• Create a new browser profile (Microsoft Edge) - Optional</h1>
It's always good to have a separate browser profile for your work and for workshops like this. This way you can keep all of your credentials separate and not have to worry about logging out of your personal / work accounts. In the example we will be using Microsoft Edge, <b>instead you can also use any other Browser, just make sure to use a profile where you are not logged in with any other Microsoft Account!</b>

<h2>Create new profile</h2>

1. Open Microsoft Edge

2. Click on the profile icon (picture) on the top right corner

3. Hover over **Other profiles** and then select select **Set up a new work or school profile**
   <img width="568" height="336" alt="image" src="https://github.com/user-attachments/assets/cc3e4206-885e-4178-9512-fbe65cb59db5" />

This will then open up a new browser window on your taskbar.

4. Right click the Edge icon on your task bar and select Pin that browser window to your taskbar
   <img width="314" height="172" alt="image" src="https://github.com/user-attachments/assets/f8a9f3e4-36f1-4925-b902-4acb1d90f041" />


5. In the new browser window, select **Start without your data**
   <img width="3730" height="2025" alt="image" src="https://github.com/user-attachments/assets/e9f77bf0-7adc-4d24-8925-c89e3b9b919d" />

6. Then select **Confirm and start browsing**.
   <img width="3720" height="2008" alt="image" src="https://github.com/user-attachments/assets/08100aca-aade-4e04-b69d-f4605701f0c4" />

It may prompt you to configure your new browser theme. If this happens, just select Next and then Finish.
