Once you're signed in Teams, navigate to **Apps**, located in the left-hand side navigation. Click on it.

<img width="956" height="454" alt="Image" src="https://github.com/user-attachments/assets/48d34ffb-3501-4c93-ab74-f09efa1b0f60" />

If you know the name of the Copilot Studio agent you're about to test, you can make use of the **Search** option. Otherwise, you can find it under the category **Built with Power Platform**. Once you have located it, click on it. 

<img width="953" height="327" alt="Image" src="https://github.com/user-attachments/assets/ad6cdc91-73dd-42b4-be3f-2dc68c1ee27a" />

In the next step, you'll be prompted by a dialogue box to add this agent to your Teams repository of tools. Click on **Add**. After a few seconds, you'll be informed that the agent was added successfully. Click **Open** to start interacting with it.
